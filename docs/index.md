# Lanternkeeper: Emberfall

A calm, ritual-based mobile companion game for solo indie developers.

A red panda named Aetherling walks alongside you through a world called Emberfall, offering emotion check-ins, micro-quests, and quiet encouragement. Never pressure.

## What is this site?

This site is the living reference for everything Lanternkeeper: Emberfall. Three sections:

- **Game Design** — The pillars, voice, and mechanics that define what this game is. Read this before making any change that touches how the game feels.
- **Dev Reference** — Architecture, conventions, contribution guide, and workflow docs. The practical how-to for building and maintaining the codebase.
- **What's Changed** — A human-readable narrative of every meaningful change. Generated from the CHANGELOG, updated weekly.
